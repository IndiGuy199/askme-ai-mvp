# Audio Files

This directory should contain:

- `chime.mp3` - A soft, calming chime sound (500ms-1s duration) that plays when transitioning between steps

You can use a free sound from:
- https://freesound.org/ (search for "soft chime" or "bell")
- https://pixabay.com/sound-effects/ (meditation bells)

Recommended characteristics:
- Duration: 0.5-1 second
- Format: MP3
- Volume: Moderate (not jarring)
- Tone: Soft, calming (like a singing bowl or meditation bell)

If you don't want audio cues, the app will continue to work - audio is optional and users can toggle it off.
